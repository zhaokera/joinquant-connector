## 策略概述

### 策略名称
**价值+动量混合量化策略 (Value + Momentum Hybrid Strategy)**

### 策略目标
| 指标 | 目标值 |
|------|--------|
| 年化收益率 | 25%+ |
| 最大回撤 | <20% |
| 夏普比率 | >1.0 |
| 波动率 | 15-20% |
| 胜率 | 55%+ |

### 策略特色
- ✅ **价值60% + 动量40%**：融合价值投资与趋势跟踪
- ✅ **双轮动机制**：行业轮动 + 风格轮动
- ✅ **动态仓位管理**：基于波动率的目标波动率控制
- ✅ **A股特色适配**：考虑T+1、涨跌停、政策驱动

## 理论基础

### 1. 经典量化理论

#### Grinold & Kahn (《量化交易：如何建立自己的算法交易事业》)
- **主动比率公式**：IR = IC × √BR
- **组合优化**：alpha + risk budgeting
- **风险調整收益**：最大化信息比率

#### Fama-French因子模型
| 因子 | 说明 | 权重 |
|------|------|------|
| Size | 大市值 vs 小市值 | 15% |
| Value | 价值股 vs 成长股 | 25% |
| Momentum | 动量股 vs 反转股 | 20% |
| Quality | 高质量 vs 低质量 | 20% |
| Volatility | 低波动 vs 高波动 | 20% |

#### Jegadeesh & Titman (动量策略)
- **短期动量**（1-4周）：继续效应
- **中期动量**（3-12月）：继续效应
- **长期动量**（3-5年）：反转效应

### 2. 价值投资理论

#### 邱国鹭《投资中最简单的事》- 四要素
1. **估值**：PE/PB/PEG评估安全边际
2. **品质**：ROE/毛利率/自由现金流
3. **时机**：市场情绪+政策导向
4. **仓位**：根据确定性动态调整

#### 李杰《股市进阶之道》- 能力圈理论
1. **好生意**：商业模式清晰，现金流稳定
2. **好公司**：管理层优秀，治理结构完善
3. **好价格**：安全边际充足

## 核心模块

### 1. 股票池筛选 (value_momentum_strategy.py)

#### 第一层：基础筛查
```python
# 条件
- 排除ST股票
- 排除停牌股票
- 上市时间 > 120天
- 市值 > 50亿（避免小市值风险）
```

#### 第二层：财务质量筛查
```python
# 条件
- ROE > 0（盈利）
- 资产负债率 < 80%
- 连续盈利记录
```

#### 第三层：动量筛查
```python
# 条件
- 1个月收益率排名前70%
- 3个月收益率排名前70%
```

### 2. 混合打分系统

#### 价值因子 (60%权重)
```python
价值得分 =
    25% × 估值得分（PE/PB/PS）
  + 25% × 品质得分（ROE/毛利率）
  + 10% × 负债率得分
  + 10% × 增长得分
  + 10% × 现金流得分
  + 20% × 综合评分
```

#### 动量因子 (40%权重)
```python
动量得分 =
    30% × 短期动量（1个月）
  + 35% × 中期动量（3个月）
  + 15% × 长期动量（12个月，含反转）
  + 20% × 成交量Ratio
```

#### 综合得分
```python
综合得分 = 60% × 价值得分 + 40% × 动量得分
```

### 3. 行业轮动 (rotation_strategy.py)

#### 行业评分公式
```python
行业得分 =
    35% × 景气度
  + 25% × 估值
  + 25% × 动量
  + 15% × 政策支持
```

#### 四大维度
1. **景气度**：营收增速、ROE、毛利率趋势
2. **估值**：行业PE/PB分位数
3. **动量**：行业指数走势
4. **政策**：政策支持力度（A股特色）

### 4. 风格轮动 (Fama-French扩展)

#### 风格评分
| 风格 | 说明 | 阈值 |
|------|------|------|
| Size | 大盘/小盘 | 0.55 |
| Value | 价值/成长 | 0.55 |
| Momentum | 动量/反转 | 0.55 |
| Quality | 高质量/低质量 | 0.55 |

### 5. 动态仓位管理 (risk_management.py)

#### Kelly公式变体
```python
仓位 = (目标波动率 / 个股波动率) × (1 / √持仓数) × 风险因子 × 市场因子

其中：
- 目标波动率：18%
- 风险因子：根据市场状态0.7-1.2
- 市场因子：牛1.15, 熊0.75, 震荡1.0
```

#### 波动率目标控制
```python
if 组合波动率 > 目标波动率 × 1.5:
    触发降仓警告

if 回撤 > 15%:
    触发风险控制
```

### 6. 风险控制

#### 止损机制
- **固定止损**：亏损12%触发
- **移动止损**：回撤8%触发
- **硬止损**：无论什么原因，达到即止损

#### 止盈机制
- **固定止盈**：盈利30%触发
- **动态止盈**：根据波动率调整

#### 流动性监控
- 日均成交额 > 100万元
- 成交量波动率 < 200%

## 风险提示

### 历史回撤模拟
| 回撤水平 | 频率 | 持续时间 |
|----------|------|----------|
| 5-10% | ~6个月 | 1-2周 |
| 10-15% | ~1年 | 1-2月 |
| 15-20% | ~2-3年 | 2-4月 |
| >20% | ~5年+ | 4月+ |

### 黑天鹅防护
- CVaR监控（95%分位）
- 极端波动警告
- 动态降仓机制
- 现金储备要求

### 策略失效场景
1. **极端value股**：市值持续萎靡
2. **流动性危机**：所有资产相关性飙升
3. **政策剧变**：行业监管重大变化
4. **模型过拟合**：历史数据回测有效，实盘失效

## 常见问题

### Q1: 为什么价值+动量混合？
**A**: 价值投资长期有效但回撤大，动量策略短期有效但易回调。混合可平衡风险收益。

### Q2: 调仓频率为什么是20天？
**A**: 过于频繁增加交易成本，过少错过机会。20天是收益和成本的平衡点。

### Q3: 如何应对A股T+1？
**A**: 采用收盘前调仓，确保当日下单。止损采用条件单实现T+0效果。

### Q4: 行业轮动有效吗？
**A**: 在政策驱动的A股市场，行业轮动显著增强收益。建议每20天调整。
